# e-plantShopping - Paradise Nursery E-Commerce Application

A dynamic React frontend application for an online plant nursery where users can browse houseplants across multiple categories, add items to a shopping cart, and manage quantities dynamically using Redux Toolkit state management.

## Project Structure
- `App.jsx` - Root layout wrapper featuring conditional page rendering and Redux Providers.
- `AboutUs.jsx` - Core company mission detail section.
- `ProductList.jsx` - Dynamic product display grid showcasing categories, thumbnails, and interactive add-to-cart status.
- `CartItem.jsx` - Shopping cart details calculation table including quantity modifications and item removals.
- `CartSlice.jsx` - Centralized Redux state slice capturing item additions, subtractions, and deletion actions.
- `App.css` - Global application layout, responsive spacing rules, and hero background overlay alignments.
